# Perfume Shop

### 개요
#### Maven 기반 Spring-MVC Framework를 사용하여 전반적인 홈쇼핑 사이트의 기능과 view 구현 

### 적용 기술
1. Language : JAVA
2. Framework : Spring, MyBatis, Bootstrap(v5.1.3)
3. Server : Apache Tomcat v9.0
4. DB : Oracle 11g
5. Web : CSS, Tiles, HTML5, JQuery, JSP, Ajax
6. Open-API :  Daum 우편번호 서비스
